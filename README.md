# ML Hub Projects

Welcome to the ML Hub Projects repository! This repository contains a structured series of 10 machine learning and deep learning projects designed to take you from foundational neural networks to advanced models in computer vision, natural language processing, and reinforcement learning.

## Repository Structure

The projects are organized sequentially as follows:

*   **Project 1:** Introduction to Perceptron, Neural Networks & PyTorch
*   **Project 2:** Image Classification with CNN & Transfer Learning
*   **Project 3:** Sentiment Analysis with RNN & LSTM
*   **Project 4:** Transformers for Text and Image
*   **Project 5:** Generative Modeling with VAE & GANs
*   **Project 6:** Language Modeling with GPT
*   **Project 7:** Object Detection with YOLO
*   **Project 8:** Reinforcement Learning: Policy & Value Iteration
*   **Project 9:** Q-Learning & DQN in Gymnasium
*   **Project 10:** Deep Reinforcement Learning with Policy Gradients

Each project is contained within its own dedicated folder. Inside each folder, you'll find a Jupyter Notebook (where applicable) and a specific `README.md` detailing the objectives, dataset, and concepts covered.
