# Project 5: Generative Modeling with VAE & GANs

## Objective
Generate new, realistic data by learning the underlying distributions behind it.

## Dataset
**MNIST:** Handwritten digit images used to train models that synthesize new data samples.

## Algorithms & Techniques
- Variational Autoencoders (VAE)
- KL Divergence
- Generative Adversarial Networks (GANs)
- Sampling from a Generative Model

## What You'll Implement
Build VAE and GAN architectures in PyTorch, modeling latent space and performing adversarial training.

## Key Learnings & Takeaways
Hands-on experience creating synthetic images with two fundamentally different generative approaches.
