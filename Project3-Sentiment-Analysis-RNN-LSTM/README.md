# Project 3: Sentiment Analysis with RNN & LSTM

## Objective
Model sequential text data to classify sentiment effectively.

## Dataset
**Amazon, IMDb, and Yelp reviews:** Text review datasets labeled with positive and negative sentiment for binary sentiment classification.

## Algorithms & Techniques
- Tokenization & Embeddings
- Recurrent Neural Networks (RNN)
- Long Short-Term Memory (LSTM)
- Vanishing Gradient Fix

## What You'll Implement
Build and compare a basic RNN against an LSTM for sentiment classification using PyTorch.

## Key Learnings & Takeaways
Clear intuition for sequence modeling and understanding how to handle long-range dependencies in text.
