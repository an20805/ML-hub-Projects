# Project 7: Object Detection with YOLO

## Objective
Run real-time object detection, then fine-tune a model to spot a custom object class.

## Dataset
**COCO (pre-trained) + Custom Pothole dataset:** 80-class pre-trained weights, fine-tuned on a labeled pothole dataset.

## Algorithms & Techniques
- YOLOv8 (Ultralytics)
- Transfer Learning
- Confidence Thresholding
- Object Detection

## What You'll Implement
Run pre-trained inference on images and video, then fine-tune YOLOv8 on a custom dataset and evaluate using training curves and a confusion matrix.

## Key Learnings & Takeaways
Deploy a custom-trained, real-time object detector on your own use case.
