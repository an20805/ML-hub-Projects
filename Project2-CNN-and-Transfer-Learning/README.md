# Project 2: Image Classification with CNN & Transfer Learning

## Objective
Classify images by learning how Convolutional Neural Networks (CNNs) capture spatial and visual structure.

## Dataset
**CIFAR-10:** Labeled photos of 10 classes used to train and fine-tune image classifiers.

## Algorithms & Techniques
- Convolution & Pooling
- Foundation Models
- Convolutional Neural Networks (CNNs)
- Transfer Learning

## What You'll Implement
Build a CNN from scratch in PyTorch, then fine-tune a pre-trained ResNet model on a new dataset.

## Key Learnings & Takeaways
Practical skill in designing CNNs, using pre-trained models, and working with image data.
