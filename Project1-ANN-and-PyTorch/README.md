# Project 1: Introduction to Perceptron, Neural Networks & PyTorch

## Objective
Build neural networks from first principles and understand the underlying mathematics that drive them.

## Dataset
**Simple tabular / synthetic datasets:** Small, clean datasets ideal for tracing every computation by hand for classification tasks.

## Algorithms & Techniques
- Perceptron
- Multi-Layer Perceptron (MLP)
- Backpropagation
- Gradient Descent

## What You'll Implement
Code forward & backward propagation and activation functions in raw NumPy, then rebuild the same network using PyTorch's `nn.Module`.

## Key Learnings & Takeaways
A rock-solid grip on Artificial Neural Network (ANN) fundamentals, plus your first working PyTorch model.
