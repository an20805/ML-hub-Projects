# Project 8: Reinforcement Learning - Policy & Value Iteration

## Objective
Understand the fundamentals of Reinforcement Learning through Dynamic Programming and environment planning methods.

## Dataset
**OpenAI Gym Grid-based Environments:** Discrete environments for learning planning and optimal decision making using known environment dynamics.

## Algorithms & Techniques
- Markov Decision Processes (MDP)
- Bellman Equations
- Policy Iteration
- Value Iteration

## What You'll Implement
Model MDPs and implement Policy & Value Iteration algorithms to solve grid-world problems.

## Key Learnings & Takeaways
A strong theoretical foundation and hands-on implementation of classical reinforcement learning algorithms.
