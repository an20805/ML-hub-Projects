# Project 9: Q-Learning & DQN in Gymnasium

## Objective
Understand model-free Reinforcement Learning by directly learning optimal behaviors without prior knowledge of the environment.

## Dataset
**OpenAI Gym Classic Control Environments:** Interactive control tasks for learning through trial-and-error.

## Algorithms & Techniques
- SARSA
- Q-Learning
- Deep Q-Networks (DQN)
- Experience Replay

## What You'll Implement
Implement tabular Q-Learning and extend it to Deep Q-Networks for solving control tasks in Gymnasium.

## Key Learnings & Takeaways
Understand how intelligent agents learn optimal behavior directly from interaction with their environment.
