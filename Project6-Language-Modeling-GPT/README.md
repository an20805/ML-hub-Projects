# Project 6: Language Modeling with GPT

## Objective
Implement a GPT-style Transformer from scratch for autoregressive text generation.

## Dataset
**Cornell Movie Dialogs Corpus:** Movie dialogue conversations used to train an autoregressive language model for next-token prediction.

## Algorithms & Techniques
- Causal Self-Attention
- RMSNorm
- Decoder-only Transformer
- Text Generation

## What You'll Implement
Implement and train a GPT-style decoder-only Transformer from scratch for next-token prediction and text generation.

## Key Learnings & Takeaways
Develop a practical understanding of how modern large language models (LLMs) are built, trained, and used for text generation.
