# Project 4: Transformers for Text and Image

## Objective
Model relationships across sequences and images using attention mechanisms within the Transformer architecture.

## Dataset
**IMDb Reviews + CIFAR-10:** IMDb movie reviews for sentiment classification and CIFAR-10 images for Vision Transformer fine-tuning.

## Algorithms & Techniques
- Self-Attention
- Multi-Head Attention
- DistilBERT
- Vision Transformer (ViT)

## What You'll Implement
Fine-tune DistilBERT for sentiment classification and a pre-trained Vision Transformer (ViT) for CIFAR-10 image classification.

## Key Learnings & Takeaways
Deep understanding of how Transformer architectures can be applied to solve both text and vision tasks.
